Part 1

As a software engineer for a startup, you're in charge of writing software to generate badge numbers and passcodes for the flood of new employess! Sounds like a big task, but good news! Since you're fresh out of college, senior developers at the company have set you up with an existing library from Exceptional Software LLC. As with most libraries, you aren't given the source code, but you have the header file and a compiled binary. You'll have to implement code for your company's application using these existing utilities, and being sure to catch all of their mysterious exceptions.

Complete the function GenerateEmployeeCredentials

Define the following function:

std::string GenerateEmployeeCredentials(const std::string& employee_name,
                                        int office_no);

This function returns a string of the form "[Employee Name] can access office number [Office Number] with Badge Number [Badge Number] and Passkey [Secret Code]."

Your implementation must call both existing functions provided in exceptional.hpp. When a seed is required, use office_no + 5.

Important notes:

  • Call ExceptionalEmployeeBadgeNumber before ExceptionalSecretCode and catch exceptions for each function independently.
  • You must explicitly catch the following exceptions: std::invalid_argument, std::out_of_range, std::runtime_error, std::logic_error. When one of these exceptions is thrown, your function should instead return a string of the form: "[Offending Function] threw [Exception Name] with message: [Exception Text]". For example, "ExceptionalSecretCode threw std::logic_error with message: Didn't think of that!"
  • You should also include a catch-all handler, which if triggered should cause your function to return "[Offending Function] caused an unknown error."

Part 2

Now that you've shown your ability to effectively use other people's code, your boss decides that your turn to write the library. They want a super secure password each day that is generated based on a set of instructions and the date.

You're given two helpful structs that package information about the data and instructions for generating the password.

The Date struct stores the month, day, and year.

struct Date {
  int month;
  int day;
  int year;
};

The Instruction struct stores both the operation and an operand to apply it to. A valid operation can be any of the characters {'+', '-', '/', '*', '%'} which corresponds to the same arithmetic operator. The operand can be either the strings "month", "day", or "year" which would be replaced by the appropriate value from a Date object or an integer stored as a string.

struct Instruction {
  char operation;
  std::string operand;
};

First, work on the helper function IsValidDate which checks if a given date is valid. Since all the values are represented as their integer counterparts, ensure that the following:

  • The month is between 1 and 12.
  • The day must be valid based on the length of the month.
  • For February, make sure to keep leap years in mind and remember that centuries are not leap years unless divisible by four hundred.
  • Ensure that the year is at least 1970.

bool IsValidDate(Date date);

Next, work on the ExecuteInstruction function which given an input integer follows an instruction to apply an arithmetic operation to it and substituting in the operand values using the date as per the instruction. You should return the result as

<value> <operation> <operand>

You may assume that the date is valid, but you must check for errors in the instruction. If the operation or operand is invalid as specified above, throw an invalid_argument exception. Also check that the arithmetic operation is valid. If performing a division or modulo by zero, throw a domain_error exception. We also want to ensure that the numbers don't get too large, so if the absolute value of the output is a million or more, throw a range_error exception.

int ExecuteInstruction(Date date, Instruction instruction, int value);

Finally, put all of this together in the function PasswordFromDate. This function should check if the given date is valid and throw an illegal_argument exception if not. Then, using the value from salt as an initial value, iterate through the instructions in procedure and execute them step by step. Make sure to use the output of the previous step as the new input for calling ExecuteInstruction. Once all the instructions are complete, return the final result.

int PasswordFromDate(Date date,
                     const std::vector& procedure,
                     int salt);
